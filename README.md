
# Daily Stoic

A progressive web app for daily stoic philosophy quotes with 
Wikipedia integration and progressive web aplication.

[View Live](https://adiilegblower.github.io/Stoic-Day/)

## ✨ Features
- Random stoic quotes fetched from custom Netlify JSON API
- Philosopher images fetched from Wikipedia API with disambiguation handling
- Audio narration using Web Speech API
- Share quotes as images using html2canvas
- Bookmark favourite quotes with local storage persistence
- Navigation history — browse previously viewed quotes
- Progressive Web App — installable on desktop and mobile
- Image caching for fast subsequent loads

## 🛠️ Built With
- Vanilla JavaScript (ES6 Modules)
- HTML5 / CSS3
- Wikipedia REST API
- Custom JSON API hosted on Netlify
- Web Speech API
- html2canvas

## 🚀 Running Locally

   git clone https://github.com/AdiiLegBlower/Stoic-Day

## 📚 What I Learned

- PWA implementation with service workers and web manifest
- Image caching with Map for performance optimization
- Web Speech API for text to speech
